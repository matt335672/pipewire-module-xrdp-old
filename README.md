# pipewire-module-xrdp

Thanks to @Hiero32 who contributed this module.

This module allows xrdp to generate sound on a pipewire-based system.

Pipewire versions 0.3.58 and later are supported.

This is the original version of this repository, before it was copied
to https://github.com/neutrinolabs/pipewire-module-xrdp

Please update all references to this repository to the above URL. This
repository will be deleted at some point in the future.
